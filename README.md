# My UX Portfolio

I made this portfolio from scratch using the following:

## Tech stack

- [Tailwindcss](https://tailwindcss.com/)
- [Babylon.js](https://www.babylonjs.com/)

## Design tools
- [Canva](https://www.canva.com/)
